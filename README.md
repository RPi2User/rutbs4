# rutbs4

## Dependencies

- sg3_utils (for SCSI Commands)
- openssl (encryption, checksumming)

THIS PROJEKT IS VERY WIP!

Only works on LTO drives!

Tested with LTO3 and LTO8 Drives, you can gladly donate me some more drives :3

### Linux

This project is tailored for Linux. Use a Fedora, openSuse or Debian. Most production testing is done on Proxmox Debian.
All not-Linux OS are not supported and won't in future.
